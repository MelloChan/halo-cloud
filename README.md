### 服务端技术选型  

- 框架: Spring Boot、Mybatis  
- 数据库: MySQL5.7  
- 缓存服务器: Redis3.2  
- 应用服务器: Tomcat8    
- Web服务器: Nginx1.12.0    
- API风格: RESTFul  
- 开发环境: Windows7 
- 部署环境: CentOS6.6    
- 构建工具: Maven3.3.9    
- 开发工具: IDEA2018      
- API测试工具: Postman  
- 版本控制: Git
- 协作平台(需求、配置、API文档等): Teambition    
- 短信平台: 云之讯      
- 图片存储: 七牛云  
- 编程规约: 阿里巴巴Java开发手册
